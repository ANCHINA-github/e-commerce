# 轻量级PHP/JSON电商系统

## 系统简介
这是一个基于PHP和JSON文件存储的轻量级电商系统，不需要数据库支持。系统包含完整的商品展示、购物车、订单管理和后台管理功能。

## 功能特性
- ✅ 商品展示（图片、名称、描述、价格、库存）
- ✅ 购物车功能（添加、更新、删除商品）
- ✅ 库存自动管理（购买时自动扣减库存）
- ✅ 后台商品管理（添加、编辑、删除）
- ✅ 订单管理（查看历史订单）
- ✅ 图片上传和管理
- ✅ 响应式设计，适配各种设备

## 安装说明

### 1. 环境要求
- PHP 7.0 或更高版本
- 文件上传权限
- JSON扩展（通常默认启用）

### 2. 安装步骤
1. 将系统文件上传到Web服务器
2. 确保以下目录可写：
   - `data/` - 存储JSON数据文件
   - `uploads/` - 存储商品图片
3. 访问网站根目录即可使用

### 3. 目录结构
ecommerce-system/
├── index.php # 前台首页
├── admin.php # 后台管理
├── cart.php # 购物车
├── process_order.php # 订单处理
├── config.php # 配置文件
├── data/
│ ├── products.json # 商品数据
│ └── orders.json # 订单数据
├── uploads/ # 商品图片
└── assets/ # 静态资源

## 使用指南

### 前台功能
1. **浏览商品**：访问首页查看所有商品
2. **添加购物车**：点击"加入购物车"按钮
3. **管理购物车**：访问购物车页面修改商品数量或删除商品
4. **结算订单**：在购物车页面点击"结算订单"

### 后台管理
1. 访问 `/admin.php` 进入管理后台
2. **商品管理**：查看、编辑、删除现有商品
3. **订单管理**：查看历史订单记录
4. **添加商品**：填写商品信息并上传图片

## 数据存储
系统使用JSON文件存储数据：
- `products.json`：存储所有商品信息
- `orders.json`：存储所有订单记录

## 安全注意事项
1. 在生产环境中，建议为后台管理添加密码保护
2. 定期备份 `data/` 目录下的JSON文件
3. 确保 `uploads/` 目录有正确的文件上传限制
