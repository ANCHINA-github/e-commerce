# 轻量级PHP/JSON电商系统

## 系统简介
这是一个基于PHP和JSON文件存储的轻量级电商系统，不需要数据库支持。系统包含完整的商品展示、购物车、订单管理和后台管理功能。

## 功能特性
- ✅ 商品展示（图片、名称、描述、价格、库存）
- ✅ 购物车功能（添加、更新、删除商品）
- ✅ 库存自动管理（购买时自动扣减库存）
- ✅ 后台商品管理（添加、编辑、删除）
- ✅ 订单管理（查看历史订单）
- ✅ 图片上传和管理
- ✅ 响应式设计，适配各种设备

## 安装说明

### 1. 环境要求
- PHP 7.0 或更高版本
- 文件上传权限
- JSON扩展（通常默认启用）

### 2. 安装步骤
1. 将系统文件上传到Web服务器
2. 确保以下目录可写：
   - `data/` - 存储JSON数据文件
   - `uploads/` - 存储商品图片
3. 访问网站根目录即可使用
